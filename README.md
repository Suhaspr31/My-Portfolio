# 🌌 Cinematic Personal Portfolio - Suhas P R

A high-performance, cinematic, and fully interactive personal portfolio designed to showcase software engineering excellence with a focus on AI/ML and Full-Stack Development.

![Portfolio Preview](https://github.com/Suhaspr31/portavia-react/raw/main/public/preview.png)

## 🚀 Experience the Motion
This portfolio is built with a **"Cinematic Minimalism"** philosophy, focusing on fluid, hardware-accelerated transitions and immersive user interaction.

### ✨ Key Features
- **Cinematic Entrance 🎬**: High-end screen reveals and staggered character-masking animations upon entering the site.
- **Interactive Apple-Style Dock 🍏**: A custom-calibrated magnification dock integrated directly into the top navigation bar.
- **Project Parallax Index 📂**: Smooth, high-fidelity project stacking effect with detailed, modal-driven deep dives.
- **Animated Branding 🔡**: Dynamic, cycling logo initials ('S', 'P', 'R') that react with sophisticated fade transitions.
- **Certified Excellence Accordion 🏆**: A hardware-accelerated, fluid certification showcase with high-vibrancy image processing.
- **Magnetic Footer UI 🧲**: A staggered, deliberate reveal of professional contact info with spring-loaded physical social elements.
- **Full-Screen Zoom Parallax 🔍**: Immersive, multi-layered visual depth experience for all key projects.

## 🛠️ Technology Stack
- **Core:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/) (Lightning-fast dev & build)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict type-safety)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Custom design system & glassmorphism)
- **Motion:** [Framer Motion](https://www.framer.com/motion/) (State-of-the-art physics & animations)
- **Smooth Scroll:** [@studio-freight/lenis](https://github.com/darkroomengineering/lenis) (High-fidelity scrolling experience)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)

## 🚦 Getting Started

### 1. Installation
```bash
# Clone the repository
git clone https://github.com/Suhaspr31/portavia-react.git

# Navigate to project director
cd portavia-react

# Install dependencies
npm install
```

### 2. Development
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 🛰️ How to Push to GitHub
Follow these steps to upload your changes to GitHub:

1. **Initialize Git (if first time):**
   ```bash
   git init
   git remote add origin https://github.com/Suhaspr31/your-repo-name.git
   ```

2. **Stage your changes:**
   ```bash
   git add .
   ```

3. **Commit with a professional message:**
   ```bash
   git commit -m "feat: implement high-end cinematic animations and project index"
   ```

4. **Push to the main branch:**
   ```bash
   git push -u origin main
   ```

## ☁️ Deployment Guide (Free)

### Option 1: Vercel (Recommended)
1. Sign up/Login at [Vercel.com](https://vercel.com/).
2. Click **"New Project"**.
3. Import your GitHub repository.
4. Vercel will auto-detect **Vite**. Click **"Deploy"**.
5. Your site is live in < 1 minute!

### Option 2: Netlify
1. Go to [Netlify.com](https://www.netlify.com/).
2. Drag and drop your `dist` folder (after running `npm run build`) or connect your GitHub repo.
3. Your site is ready!

### Option 3: GitHub Pages
1. Install the `gh-pages` package: `npm install gh-pages --save-dev`.
2. Add a `homepage` field to `package.json`.
3. Run `npm run deploy`.
